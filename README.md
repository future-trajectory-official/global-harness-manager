# global-harness-manager

AI エージェントの制御・運用を司る「ハーネス（Harness）」のグローバル管理リポジトリです。
2026年時点の **ハーネス・エンジニアリング（Harness Engineering）** の思想に基づき、プロンプトやコンテキストを超えた、システムレベルでのエージェント制御を実現します。

## 目的

本リポジトリは、複数のプロジェクトで横断的に利用される AI エージェント用の「道具（Skills）」「制約（Rules）」「手順（Workflows）」を一括管理し、エージェントの挙動の信頼性と再現性を高めることを目的としています。

## 管理対象（コアレイヤー）

本リポジトリでは、以下の3つの主要なレイヤーを管理します。

*   **Tools Layer (`.agents/skills/`)**: エージェントが実行可能な専門スキルの登録と管理。
*   **Constraints Layer (`.agents/rules/`)**: エージェントの挙動を縛るグローバルな制約や文脈の定義。
*   **Feedback Layer (`.agents/workflows/`)**: 複雑なタスクの自動化や、自己修復・検証のループ手順の定義。

## 構成

- `scripts/`: ハーネスを各環境に適用し、資産を接続するための制御スクリプト群。
- `~/.gemini/antigravity/skills.txt`: 本リポジトリをグローバル検索パスとして登録するための設定。
