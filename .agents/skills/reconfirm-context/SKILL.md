# reconfirm-context

このスキルは、チャットの履歴が長くなった際や、エージェントが自身の役割（Rules）や利用可能なツール（Skills）を正しく認識できなくなっている疑いがある場合に、「現在の自己像」をリフレッシュするためのものです。

ワークスペース（ローカル）のルールと、システムに登録されたグローバルなスキルの概要を最小限のコンテキスト消費で再インデックスします。

## 手順

// turbo-all
1. **メタデータの抽出実行**
   - 以下の動的なコマンドを実行し、登録されたスキルパスから `index-capabilities.sh` を見つけて実行する。
   ```bash
   # 登録済みスキルパスからスクリプトを自動探索して実行
   $(cat ~/.gemini/antigravity/skills.txt | xargs -I {} find {} -maxdepth 3 -name index-capabilities.sh | head -1)
   ```

2. **自己認識の同期**
   - 出力された各ルール（Roles）とスキル（Capabilities）の内容を読み込み、「現在のタスクにおいて、どの役割として振る舞うべきか」および「どのスキルが活用可能か」を改めて自分自身とユーザーに宣言する。

3. **コンテキストのクリーンアップ確認**
   - ここで得た「最小限の概要」をベースに、不要な深掘りを避け、本来のタスクの目的に集中する。

※ 注意: このスキルは情報の「発見」ではなく「再定義・再認識（Indexing）」を目的としています。詳細なコードや文書の確認が必要な場合は、適宜ツールを使用して個別に取得してください。
