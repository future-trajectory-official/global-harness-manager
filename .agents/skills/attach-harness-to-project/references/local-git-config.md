# Technical Spec: Local Git Configuration

プロジェクトごとの Git ユーザー情報設定に関する仕様です。

## 1. プロジェクト・ローカル設定
本スキルは、対象リポジトリの `.git/config`（ローカル設定）に対して以下の値を書き込みます。
- `user.name`
- `user.email`

## 2. グローバル設定の保護
グローバルな `git config --global user.name` 等は変更しません。プロジェクトごとに異なるアイデンティティを「装着」することで、複数のプロジェクトが混在する環境でも、コミットログの著作者情報が混ざることを防ぎます。

## 3. identities.txt との同期
設定値は必ず `config/identities.txt` から取得されます。管理外のアカウント情報が勝手に設定されることはありません。

## 4. 実行の冪等性
既に正しい設定が書き込まれている場合は、重複した処理を行いません。
