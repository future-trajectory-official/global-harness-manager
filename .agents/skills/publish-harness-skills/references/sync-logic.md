# Technical Spec: Skill Synchronization Logic

ローカルスキルをグローバルディレクトリへ同期する際の技術的な仕様です。

## 1. 同期エンジン (rsync)

同期には `rsync` を使用し、`.gitignore` を考慮した差分コピーを行います。

- **`--delete` オプション**:
  同期元で削除されたファイルは同期先でも削除され、常にクリーンな状態を維持します。
- **`.gitignore` の尊重**: スキルディレクトリ内に `.gitignore`
  が存在する場合、そこに記述されたファイル（`bin/`, `tmp/` 等）は同期対象から自動的に除外されます。

## 2. 共通の除外ルール

`.gitignore` の有無にかかわらず、以下の項目は常に同期から除外されます。

- `.git/` ディレクトリ
- `.DS_Store`
- `__pycache__/` 等のキャッシュファイル

## 3. アトミックな更新

公開先ディレクトリに同名のスキルが既に存在する場合、rsync
によって効率的に更新されます。これにより、大規模なスキルセットであっても高速に同期を完了できます。
