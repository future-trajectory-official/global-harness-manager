# Configuration: Publish Targets for Skills

ワークスペース内のスキルをグローバルディレクトリに同期するための `config/publish-targets.md` の記述形式です。

## 1. 記述フォーマット (Markdown 形式)
本ファイルは Markdown 形式で管理されており、`## ` (H2ヘッダー) を使用して同期対象を指定します。

- **有効化**: 同期したいスキル名の行頭に `## ` を記述します (例: `## reconfirm-context`)。
- **無効化**: 行頭の `## ` を削除してプレーンテキスト化、またはコメントアウトすることで、そのスキルは同期対象から外れます。

## 2. 実行の仕組み
スクリプトは `grep '^## '` を使用して、H2ヘッダーで始まる行からスキル名のみを抽出し、同期処理を実行します。

## 3. 注意事項
- スキル名は `.agents/skills/` に実在するディレクトリ名と一致させてください。
- 説明文（**説明**: ...）などはスクリプトによって無視されるため、自由に記述可能です。
