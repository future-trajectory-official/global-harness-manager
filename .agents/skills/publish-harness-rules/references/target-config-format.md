# Configuration: Publish Targets for Rules

共通ルール（.md）をどのプロジェクトに配信するかを定義する `config/publish-rules-targets.md` の正しい記述形式です。

## 1. 記述フォーマット (Markdown 形式)
本ファイルは Markdown 形式で管理されており、`###` (H3ヘッダー) を使用して対象を指定します。

- **プロジェクトの指定**: `## Target Projects` セクションの下に、プロジェクトの絶対パスを `###` で記述します。
- **ルールの指定**: `## Target Rules` セクションの下に、同期したいルール名（拡張子 `.md` を除いたもの）を `###` で記述します。

## 2. 同期の有効・無効の切り替え
同期対象から外したい場合は、行頭の `###` を削除してプレーンテキスト（またはコメントアウト）状態にします。スクリプトは `### ` で始まる行のみを有効なターゲットとして抽出します。

## 3. 注意事項
- ルール名は `.agents/rules/` に実在するベース名と一致させてください。
- パスは存在している必要があります。
